# Finance Tracker

Este projeto é um **rastreador financeiro simples** desenvolvido com **HTML, CSS e JavaScript**. Ele permite que o usuário adicione gastos, categorize-os (Gasolina, Comida e Compras) e acompanhe o valor acumulado de cada categoria. Os dados são armazenados utilizando **localStorage**, garantindo que as informações permaneçam salvas mesmo após atualizar ou fechar a página.

O usuário pode adicionar novos gastos, resetar o valor total e visualizar quanto foi gasto em cada categoria. O layout da interface é organizado com CSS, posicionando os botões das categorias acima dos respectivos valores exibidos, permitindo uma visualização clara e rápida dos gastos registrados.
